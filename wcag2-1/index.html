Reflow: low vision

Reflow exceptions

  • Tabular data
  • Other widgets that rely on 2-dimensional presentation (e.g. a flowchart)
  • Toolbars?
Ask questions!

Non-text Contrast: low vision

Non-text Contrast exceptions

  • Logos
  • Text alternative immediately adjacent
  • Toolbars?
Ask questions! Text must be immediately next to the graphic. Also applies to control boundaries

Text Spacing: cognitive, vision

Common misconception: you must meet this requirement in your own styles

Text Spacing exceptions

None! :)

Content on Hover or Focus: mobility, vision

Content on Hover or Focus exceptions

  • Popups that will never cover other content
  • Popups that the user actively opens (e.g. with click, enter, etc.)
Tooltips! Zoom!

Status Messages: low vision

Status Message exceptions

  • UI changes that cause a forced context/focus change
  • Immediate and successful actions (e.g. opening an accordion does not need a status message)
  • Too many or too frequent announcements can be distracting, use with care
Ask questions!
